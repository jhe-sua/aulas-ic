-------------Software-------------

BIOS: Basic Imput/Output System
	É o software embutido no Hardware que liga a placa mãe aos outros hardwares

Sistemas Operacionais:
	É um software mais avançado que permite realizar mais trabalhos, é possível utiliza-lo graças a BIOS

Drivers:
	São os que informam ao SO as especificações do Hardware que é possível encontrar através da BIOS para que funcione corretamente


Hardware > BIOS > SO > Drivers > Aplicativos > Usuário
